# Loom Studio 使用指南

## 改进内容

### 1. 全屏拓扑图
- 拓扑图现在占据整个屏幕空间
- 不再局限于左侧小面板
- 右侧显示节点详情面板（点击节点时）

### 2. 节点状态可视化
- **绿色节点** = Agent 节点
- **蓝色节点** = Tool 节点  
- **橙色节点** = Crew 节点
- **黄色边框** = 正在执行
- **绿色边框** = 已完成
- **红色节点** = 错误状态

### 3. 节点详情面板
点击任意节点可以查看：
- **节点状态**：idle / running / completed / error
- **输入事件**：显示最近 5 个输入事件，包含完整数据
- **输出事件**：显示最近 5 个输出事件，包含完整数据
- **事件时间戳**：每个事件的时间信息

### 4. 实时执行追踪
- 系统会自动高亮当前正在执行的节点
- 边（连接）也会高亮显示数据流向

## 视图说明

### Topology 视图
- 显示所有节点和它们之间的连接
- 节点大小和颜色表示类型
- 边框颜色表示执行状态
- 点击节点查看输入输出

### Timeline 视图
- 显示事件的时间线
- 每个节点一行
- 事件用不同颜色的点表示

### Memory 视图
- 显示节点的记忆状态
- 查看记忆的演变过程

## 提示

- **点击节点**：查看节点的详细输入输出
- **观察颜色**：快速了解节点状态
- **查看边上的数字**：表示节点间通信次数
- **实时更新**：事件会自动推送到前端，无需刷新
