# 演进路线图 (Roadmap)

基于我们的设计哲学，以下是 Loom 的未来演进规划：

## 短期目标 (Short Term v0.3 - v0.4)
- [ ] **可视化调试器 (Loom Studio)**: 将不可见的自组织过程可视化。
- [ ] **分布式强化 (Distributed Hardening)**: 提供开箱即用的 Redis/NATS 传输层实现。

## 中期目标 (Mid Term v0.5 - v1.0)
- [ ] **分形运行时 (Fractal Runtime)**: 支持动态代码生成与热重载，允许 Agent 编写并运行自己的 Tool。
- [ ] **多模态记忆 (Multimodal Memory)**: 将新陈代谢记忆扩展到图像和音频领域。

## 长期愿景 (Long Term v2.0+)
- [ ] **自我进化 Swarm (Self-Evolving Swarm)**: 能够根据任务难度，自动分裂、合并、生长的 Agent 集群。
- [ ] **通用 Agent 协议 (Universal Agent Protocol)**: 实现跨框架（Loom ↔ AutoGen ↔ LangGraph）的互操作性。
