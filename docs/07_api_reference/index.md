# API Reference

This section provides detailed technical specifications for the Loom Framework SDK.

## Core API
- **[LoomApp & Kernel](loom_app.md)**: Main entry point and event system.
- **[Nodes & Agents](nodes.md)**: The building blocks of intelligence.

## Usage Convention
Loom uses Python type hints extensively. All public APIs are asynchronous (`async def`) unless otherwise noted.
