# 安装

## 系统要求

- Python 3.9+
- pip 或 poetry

## 安装方式

### 使用 pip（推荐）

```bash
pip install loom-agent
```

### 从源码安装

```bash
git clone https://github.com/your-org/loom-agent.git
cd loom-agent
pip install -e .
```

## 验证安装

```python
import loom
print(loom.__version__)
```
