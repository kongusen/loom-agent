# Installation

## System Requirements

- Python 3.9+
- pip or poetry

## Installation Methods

### Using pip (Recommended)

```bash
pip install loom-agent
```

### Install from Source

```bash
git clone https://github.com/your-org/loom-agent.git
cd loom-agent
pip install -e .
```

## Verify Installation

```python
import loom
print(loom.__version__)
```
