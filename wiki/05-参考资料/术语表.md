# 术语表

## Axiom / 公理系统

用 `A = ⟨C, M, L*, H_b, S, Ψ⟩` 定义 Agent 结构的顶层语言。

## Harness / `Ψ`

运行时环境、边界和约束的总称。它负责提供舞台，不负责替模型做语义决策。

## Context / `C`

模型当前可见的认知界面。不是“所有信息的总和”，而是经过组织后的感知入口。

## Memory / `M`

超出当前上下文窗口的记忆系统，包括会话、持久化和语义层。

## Loop / `L*`

主执行闭环，通常写作：

```text
Reason → Act → Observe → Δ
```

## Heartbeat / `H_b`

独立感知层，用来在长时间等待或执行期间监控外部变化。

## Skill / `S`

按需加载的能力单元，可以通过工具、Plugin、MCP 等形式表现出来。

## renew

当上下文压力过高时，对上下文进行压缩、重建并继续任务的过程。

## d_max

多 Agent 递归拆解的最大深度，属于物理约束的一部分。
