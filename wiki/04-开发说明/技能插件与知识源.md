# 技能插件与知识源

新的运行时 API 已经把"画像、知识源、生态能力"拆成几组独立对象。

## AgentProfile

`AgentProfile` 定义在 `loom/api/profile.py`，是 `AgentRuntime` 的输入。

它当前包含：

- `id`
- `name`
- `config`
- `skills`
- `knowledge_sources`

示例：

```python
from loom.api import AgentProfile

profile = AgentProfile.from_preset("coding")
```
