    Installation¶

    System Requirements¶

    • Python 3.9+
    • pip or poetry

    Installation Methods¶

    Using pip (Recommended)¶

    pip install loom-agent

    Install from Source¶

    git clone https://github.com/your-org/loom-agent.git
cd loom-agent
pip install -e .

    Verify Installation¶

    import loom
print(loom.__version__)
